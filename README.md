Цей застосунок реалізує просту систему обліку відвідувань лабораторії. Користувач вводить своє ім’я та може натиснути кнопку «Увійти» або «Вийти», після чого інформація зберігається на сервері та відображається на сторінці.

Фронтенд створений на React з використанням Vite.
Основний файл: client/src/App.jsx У цьому файлі реалізовано поле введення імені (зберігається у стані name через useState); кнопки «Увійти» та «Вийти» (надсилають POST‑запит на сервер через функцію send());
відображення списку людей у лабораторії (стан list, який оновлюється після кожного запиту); завантаження початкових даних (виконується в useEffect, де робиться GET‑запит до сервера);
простенькі стилі інтерфейсу (описані в об’єкті styles у кінці файлу).

Бекенд написаний на Node.js + Express. Основний файл: server/index.js. У цьому файлі створюється сервер Express; налаштовується маршрут /api/visits; обробляються два типи запитів:
GET повертає список людей, які зараз у лабораторії; POST додає або видаляє ім’я зі списку залежно від дії (enter або exit). Дані зберігаються у масиві people.

У Visual Studio Code застосунок запускається за допомогою двох терміналів.
В одному пишемо:
cd server
node index.js

В другому:
cd client
npm run dev

І переходимо за посиланням на http://localhost:5173

За посиланням побачимо, що реалізований функціонал введення імені користувача, фіксація входу та виходу, оновлення списку людей у лабораторії в реальному часі та зручний мінімалістичний інтерфейс.

Цей застосунок може використовуватися як журнал відвідувань лабораторії, система контролю присутності, навчальний приклад роботи з React та Express.
